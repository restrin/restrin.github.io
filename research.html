My research is focused on developing and analyzing fast algorithms for large-scale computational problems arising in scientific computing, machine learning, and other fields. Below is a brief description of (some) projects that I have worked on during my PhD and during internships.

Projects

  • Smooth Exact Penalty Functions for Nonlinear Programming
    • We develop and implement smooth exact penalty functions for nonlinear programs: \begin{align*} \text{minimize}_x \qquad & f(x) \\ \text{subject to} \qquad & c(x) = 0 \\ & \ell \le x \le u \end{align*} based on a smooth penalty function originally introduced by Fletcher in 1970.
    • The Matlab solver is here, and a prototype is implemented within the ROL package in Sandia's Trilinos Library.
  • BCFLASH: A Newton-CG Trust-Region Solver for Bound-Constrained Problems
    • A Matlab solver based on TRON for solving: \begin{align*} \text{minimize}_x \qquad & f(x) \\ \text{subject to} \qquad & \ell \le x \le u \end{align*}
    • Solver can be found here.
  • Iterative Solvers for Linear Systems, Least-Squares and Least-Norm Problems
    • LSLQ and LNLQ are Krylov solvers for least-squares and least-norm problems that allow for upper-bounding the solution error at each iteration, provided spectral information on the linear system.
    • SPMR is a family of iterative solvers for non-symmetric saddle-point systems, aimed to exploit the saddle-point structure of the underlying linear system.
    • These solvers are implemented within my Github repository LinearSystemSolvers, and within the Krylov.jl package.
  • PDCO: A Primal-Dual Interior Point Solver
    • PDCO is a primal-dual interior point solver for problems of the form: \begin{align*} \text{minimize}_{x,r} \qquad & \phi(x) + \frac12 \|D_1x\|^2 + \frac12 \|r\|^2 \\ \text{subject to } \qquad & Ax + D_2r = b \\ & \ell \le x \le u, \end{align*} where \( \phi(x) \) is a convex function. Details on the original MATLAB solver is found here.
    • The solver is implemented in C++ in order to support higher precision datatypes, such as DoubleDouble, QuadDouble, Quad and BigFloat.
    • This solver is applied to multiscale linear programs arising from Metabolic Expression models.
    • PDCO is being implemented within my fork of Elemental, found here.
  • Lattice Problems
    • Several algorithms (LLL, BKZ) for lattice problems (e.g. SVP, CVP) implemented into Elemental.

Internships

  • Google Research (Summer 2017)
    • Researched new approaches to low-rank matrix completion with applications to recommendation systems (e.g. movie or music recommendations given partial ratings) and to compute word embeddings.
    • Implemented high-performance solver for various low-rank matrix completion schemes using weighted alternating least-squares in python using Numpy and Scipy.
    • Demonstrated cases where proposed low-rank completion structure outperforms traditional low-rank matrix-completion approaches for computing word embeddings.
  • Microsoft (Summer 2015)
    • Implemented features for distributed database transations in the cloud for SQL Azure as part of the Elastic Scale team.
    • Algorithms implemented from novel work conducted at Microsoft Research.
    • Feature was implemented in C++ within the SQL Server Engine, along with a design document and a test suite in C#.
    • More details to follow when feature is in public preview.
  • Microsoft (Summer 2014)
    • Worked on designing a time synchronization scheme for virtual machines throughout Azure datacenters across the world. The current state of synchronizations let machines drift up to seconds and minutes apart, making logging and causality analysis across nodes difficult.
    • Implemented prototype which achieved synchronization to within 4 milliseconds for nodes in the same datacenter. Across datacenters, sub-second synchronization was achieved.
  • Google (Summer 2013)
    • Was a software engineering intern for Gmail, working to improve the network efficiency of the request/response system between the front- and back-end of the mobile application.
    • Resulting project reduced the round-trip time of the first request/response pair by 75%.
